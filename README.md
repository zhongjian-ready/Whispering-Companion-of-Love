# 喝水提醒小程序

一个帮助用户养成良好饮水习惯的微信小程序。

## 功能特性

### 🏠 首页

- **饮水进度展示**：直观显示今日饮水进度和完成百分比
- **快速添加**：提供常用饮水量快捷按钮（200ml、300ml、500ml、800ml）
- **自定义添加**：支持手动输入任意饮水量
- **今日记录**：显示当天的饮水记录，支持删除操作
- **智能提醒**：可开启/关闭定时饮水提醒
- **鼓励语句**：根据完成进度显示不同的鼓励文字

### 📊 历史页面

- **数据统计**：显示今日饮水、本周平均、坚持天数、目标达成率
- **日期选择**：可查看任意日期的饮水记录
- **7 天趋势图**：直观展示最近一周的饮水趋势
- **成就系统**：解锁各种饮水成就，激励用户坚持

### ⚙️ 设置页面

- **目标设置**：自定义每日饮水目标（500-5000ml）
- **提醒配置**：设置提醒间隔和时间段
- **快速添加配置**：自定义快速添加按钮的饮水量
- **数据管理**：支持数据导出和清空功能
- **健康小贴士**：提供科学的饮水建议

## 技术特点

- **数据持久化**：使用微信小程序本地存储保存用户数据
- **响应式设计**：适配不同尺寸的手机屏幕
- **渐进式交互**：流畅的动画效果和用户反馈
- **智能提醒**：基于用户设置的智能提醒系统

## 使用说明

1. **首次使用**：打开小程序后会自动设置默认的饮水目标（2000ml）
2. **记录饮水**：在首页点击快速添加按钮或输入自定义数量
3. **查看历史**：在历史页面查看过往的饮水记录和趋势
4. **个性化设置**：在设置页面调整目标、提醒等个人偏好

## 开发说明

### 环境要求

- 微信开发者工具
- 微信小程序账号

### 部署步骤

1. 在微信公众平台注册小程序账号
2. 获取 AppID 并替换 `project.config.json` 中的 `appid` 字段
3. 使用微信开发者工具打开项目
4. 预览和调试功能
5. 提交审核并发布

### 数据存储

- 使用 `wx.setStorageSync()` 和 `wx.getStorageSync()` 进行数据持久化
- 主要存储键值：
  - `waterData`: 当日饮水数据
  - `settings`: 用户设置
  - `water_YYYY-MM-DD`: 历史数据（按日期）

## 健康建议

根据《中国居民膳食指南》建议：

- 成年人每日饮水量：1500-1700ml
- 运动后需额外补充水分
- 建议少量多次饮水
- 避免睡前大量饮水
